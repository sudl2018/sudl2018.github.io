sklearn的波士顿房价数据集01

sklearn 机器学习 sklearn 数据集 2018/02/01

引入包

这里我们用到两个包

  • 一个是sklearn 的自带的数据集合
  • 一个是sklearn 的线性回归

格式非常的固定

  1. 只要是自带的数据集，都是load_XXX()
  2. 数据特征都是.data;标签都是.target
  3. 定义模型，一般一句话超级简单
  4. 训练模型，都是XX.fit(特征，标签)
  5. 预测结果xx.predict(特征)
  6. 评价模型.score(特征，标签) 大小[0,1]越接近1越好

评价指标有很多种 sklearn API

输出：

LinearRegression(copy_X=True, fit_intercept=True, n_jobs=1, normalize=False)

输出：

array([30.00821269, 25.0298606 , 30.5702317 , 28.60814055])

输出：

array([24. , 21.6, 34.7, 33.4])
