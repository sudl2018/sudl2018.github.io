05_Spring Initializer 快速创建Spring Boot项目

SpringBoot SpringBoot 2018/09/25

IDE 都支持使用Spring 的创建向导快速创建项目

选择你需要的启动器，向导会联网创建Spring Boot项目

默认生成Spring Boot 项目：

  • 主程序已经ok了

  • resources 文件夹中的目录结构

    • static 保存静态资源：js css images;

    • templates 保存所有的模板页面：（默认jar包，默认不支持jsp, 但是可以使用模板引擎，freemarker,thymeleaf）

      • application.properties :Spring Boot 的默认配置文件，可以修改默认配置，直接打server. 就可以了
