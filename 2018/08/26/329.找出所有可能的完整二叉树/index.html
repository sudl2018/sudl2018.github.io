所有可能的完整二叉树

二叉树 medium 894 算法 2018/08/26

问题

找出所有可能的完整二叉树

All Possible Full Binary Trees

完整二叉树是一类二叉树，其中每个结点恰好有 0 或 2 个子结点。

返回包含 N 个结点的所有可能完整二叉树的列表。 答案的每个元素都是一个可能树的根结点。

答案中每个树的每个结点都必须有 node.val=0。

你可以按任何顺序返回树的最终列表。

注意完整二叉树不是完全二叉树

思路一（深度优先搜索尝试所有可能节点，深拷贝二叉树）

  1. 找出所有的树，那必须有很多根节点，返回是个根节点的list
  2. 我不知道结果有多少颗树，所以不能一下子列出所有根节点，所以我首先考虑：
    1. 从根节点构造树的所有合理可能
    2. dfs——从根节点开始，每次递归 都是把左右子节点放入树中的某个节点，所有可能节点都要尝试（for 循环），但是使用for 循环要考虑重复访问的问题
    3. 把每一种可能进行深拷贝后添加到结果中

思路二（动态规划）

  1. 完整二叉树的左右子树也都是完整二叉树

  2. 左子树的所有可能和右子树的所有可能的组合就是答案

  3. 从节点数是1开始构造，一直构造到N
